## 马上十年了
从毕业加入 “只做事情”型 公司开始，就不可避免滑向 “熟练搬砖工”， 这正是各厉害前辈分享时最爱说到 “我没去 xx公司 就是因为听下来就没什么意思” 的那种工作。

做了快十年了。

靓丽学历的起点和过程中的钻研我一个都没有，这种结果也是一个必然。

## 过去的 2025年
从 24年底 声网 垮台开始，整个一年都在余波中。

一开始冲着 百度 作为 云服务提供商 + 自研大模型能力 招牌进去，结果现在核心任务其实是客服，且基本上隔天就有半夜的 on call 电话进来，且未来技术掉队只会更明显；

而后找一个职级为 合伙人-1 的创业公司做 “AI Agent”，结果一句话总结：草台班子 + 打零工 + 挂羊头卖狗肉，不可能有技术进步的，还经常加班到深夜；

## 2026年 新年计划
好在年关之前找到了一份 Gen AI with global team 的工作，一段时间下来感叹这才是理想中正规的工作。

终于有时间和精力做自己想做的事情了，脱离随时回消息和赶（随时变动的）需求的恐惧，看看我这一年能做点什么吧。
