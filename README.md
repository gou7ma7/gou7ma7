## 2026年 马上十年了
过去的 2025年 可以说为了在工作十年之际转型走上正规而走了巨大的弯路。

职业生涯起点就没对，期间又没能深入钻研技术，所以为了 “追上” 只能付出更多的努力了。

## 过去的 2025年
从 24年底 声网 垮台开始，整个一年都在余波中。

一开始冲着 百度 作为 云服务提供商 + 自研大模型能力 招牌进去，结果现在核心任务其实是客服，且基本上隔天就有半夜的 on call 电话进来，以后的技术掉队之后更明显；

而后找一个职级为 合伙人-1 的创业公司做 AI Agent，结果一句话总结：草台班子 + 打零工 + 挂羊头卖狗肉，还经常加班到10点以后；

好在年关之前找到了一份 gen AI 的工作，看起来可以真正的走上正规了。
